# Student Registration and Fee Management System

A web-based application built using Java Servlets, JDBC, Oracle SQL, HTML, and CSS.  
The system allows student registration, fee submission, and record search through dynamic server-side processing.

---

## Features

### Student Registration
- Collects student details such as name, phone number, course, batch time, and address.
- Stores data in the Oracle database using JDBC and PreparedStatement.

### Fee Management
- Accepts total fee, paid fee, and balance.
- Inserts fee data into the database through a dedicated Fee servlet.

### Student Search
- Retrieves student details by name.
- Displays query results in a dynamic HTML table using ResultSetMetaData.

---

### Technologies Used
- Java Servlets (Jakarta EE)
- JDBC
- Oracle SQL Database
- HTML, CSS
- Apache Tomcat (or any Java EE supported server)
